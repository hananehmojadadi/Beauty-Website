Beauty Website

The website includes a modern design with animations, services section, packages, testimonials, and contact form.

Beautiful gradient navbar
Hero section with image and animation
About section
Services section
Packages section
Testimonials
Contact form
Fully responsive design (mobile friendly 📱)
Smooth animations and hover effects

🎨 Technologies Used:

HTML5
CSS3
Bootstrap
Responsive Design
Animations using CSS

📱 Mobile Friendly:
The website is optimized for all devices including mobile phones, tablets, and desktop screens.

🎯 Purpose:
This project is created for beauty salon, makeup artist, or skincare business websites.
